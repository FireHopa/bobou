Coloque aqui arquivos .ttf ou .otf da fonte usada nos criativos.
O renderer local vai priorizar essas fontes antes das fontes padrão do sistema.

Exemplos:
- Gotham.ttf
- Gotham-Bold.ttf
- Montserrat-Regular.otf
- Montserrat-SemiBold.otf

Se não houver fonte aqui, o sistema usa Arial / DejaVu / Liberation como fallback.
